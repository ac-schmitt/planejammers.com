Erscheinungsvermerk

Dies ist eine reine private Website zur Unterstützung einer privaten Rollenspielrunde. Veranstalter ist die fiktive Fantasy-Kanzlei Phineas, Ilominar, Fohleireih & Co.

Kontaktaufnahme

Die Quellen für die Seite befinden sich auf Github in einem öffentlichen Repository. In diesem Repository können Issues erstellt werden, die vom Betreiber der Website gelesen werden. Alternativ könnte auch eine Mail an website(at)planejammers.com funktionieren, falls sie an den Spamfiltern vorbei kommt.
